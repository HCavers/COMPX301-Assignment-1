Usage:
		java MakeRuns <Heap Size>
		
		MakeRuns takes data to sort through standard in and makes a 
		text file that contains the runs generated as output. It prints 
		the number of runs and name of the text file to standard out.
		
		
		java PolyMerge <Number of files>
		
		PolyMerge takes the number of runs and the name of the text file
		through standard in in that order. It prints the sorted file to standard 
		out.
